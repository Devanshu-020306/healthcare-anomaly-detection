---

## 📱 Super Quick Deployment (2 minutes!)

### Step 1: Go to Replit

1. Visit: **https://replit.com**
2. Sign up with GitHub (free)

### Step 2: Import Repository

1. Click **"Create Repl"**
2. Select **"Import from GitHub"**
3. Paste: `https://github.com/Devanshu-020306/healthcare-anomaly-detection`
4. Click **"Import from GitHub"**

### Step 3: Configure

1. Language: **Python**
2. Click **"Import from GitHub"**
3. Wait 30 seconds for import

### Step 4: Run!

1. Click **"Run"** button
2. Replit will:
   - Auto-install dependencies
   - Start backend
   - Give you a URL!

### Step 5: Get Your URL

Your backend is live at:
```
https://healthcare-anomaly-detection.YOUR_USERNAME.repl.co
```
